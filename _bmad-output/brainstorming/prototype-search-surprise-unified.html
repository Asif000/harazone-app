+ Clean, minimal — one element
+ Surprise Me feels like a hidden power move
- Long-press is not discoverable
~ Could add a subtle dice icon hint that invites the long-press

C. Dice in Search Bar — Surprise Me Lives in the Search

The search bar gets a dice icon next to the counter. "Search this area" pill stays as-is below. The dice is the Surprise Me trigger — it's IN the search bar because discovery IS search. Compact, no extra UI.

+ No new UI elements — fits into existing search bar
+ Dice icon is universally understood as "random"
+ Search this area pill stays untouched
- Small tap target in the search bar
~ Dice could animate/wiggle periodically to draw attention
